# 🏥 TourAid – Tourist Medical Assistance Platform

TourAid is a web application designed to help **tourists and travelers find nearby hospitals and clinics quickly**, even in cities where they face **language barriers and lack of local knowledge**.

---

## 🚀 Problem

Travelers often face serious challenges when they fall sick in unfamiliar places:

* Don’t know nearby hospitals
* Language barriers with doctors
* No clarity on consultation costs
* Difficulty in emergency situations

---

## 💡 Solution

TourAid provides a **simple and fast way to access medical help**:

* 🔍 Find nearby hospitals instantly
* 🌐 Filter by language (English support)
* 📞 One-click “Call Now” feature
* 💬 Built-in **translation support (USP)**
* 💰 Basic consultation fee visibility

---

## 🔥 Unique Feature (USP)

### 🌍 In-App Medical Translation

Instead of relying only on English-speaking doctors:

* Users can select predefined medical phrases
* The app shows translations in local language
* Enables communication even if doctor doesn’t speak English

👉 Simple, fast, and reliable solution for real-world use

## ⚙️ Features

* 🏥 Hospital listing with details
* 📞 Direct call integration (`tel:`)
* 💬 Translation feature using predefined phrases
* 📱 Responsive UI
* ⏳ Loading & error states (planned)

---

## 🔄 User Flow

1. User opens the app
2. Searches for nearby hospitals
3. Applies filters (language/type)
4. Views hospital list
5. Calls hospital OR uses translation feature

### 2. Install dependencies

